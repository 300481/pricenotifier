# pricenotifier
price notifier for cheap car fuel prices

## environment

```
GOOGLE_APPLICATION_CREDENTIALS=credentials.json # your GCP credentials file
API_KEY=00000000-0000-0000-0000-000000000002    # your tankerkoenig API key
LAT=49.833458                                   # your position lat
LNG=8.052952                                    # your position lng
RAD=6                                           # your wished radius to look for stations
PUSHOVER_TOKEN=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxx   # your pushover token
PUSHOVER_USER=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxx    # your pushover user
DAYS=14                                         # your wished prices history length to calculate good prices for
```
